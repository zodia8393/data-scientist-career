# Hiring Market Alignment

## 목표 역할

- Data Scientist
- AI/ML Product Data Scientist
- Applied AI Engineer
- LLM Evaluation Engineer
- ML Engineer

## 역량 신호

| 평가축 | 프로젝트 증거 |
|---|---|
| DS 문제정의 | 운영 의사결정과 risk/review gate 연결 |
| Applied AI | agent tool-use, guardrail, human review queue |
| LLM evaluation | task success, invalid action, evidence citation metric |
| ML Product | model/incident/impact output을 approval workflow로 연결 |
| MLE | CLI, CI, tests, trace/report artifact |
| Product DS | reviewer workload, impact claim blocker, queue priority를 metric으로 제시 |

## 채용 포지셔닝

이 프로젝트는 순수 agent demo가 아니라 운영 ML 시스템 위의 agent evaluation layer다. 전통 DS와 최신 Applied AI 사이의 연결 역량을 보여주고, Stage 3 Control Tower의 impact card까지 읽어 Product DS/MLE delivery 신호를 강화한다.
